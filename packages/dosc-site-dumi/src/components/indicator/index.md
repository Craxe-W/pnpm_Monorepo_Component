## 属性

| 属性        | 说明     | 类型     | 默认值    |
| ----------- | -------- | -------- | --------- |
| size        | 尺寸     | `number` | `16`      |
| color       | 颜色     | `string` | `#ff1a34` |
| bgColor     | 背景颜色 | `string` | `#eee`    |
| borderWidth | 粗细     | `number` | `5`       |
